# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-11-30

### Added
- Initial release of GitHub Repo Analyzer
- Repository information extraction functionality
- Beautiful popup interface with gradient design
- Support for extracting:
  - Repository owner and name
  - Repository description
  - Stars, Forks, Watching statistics
  - Programming language
  - License information
  - Topics/tags
  - Issues and Pull Requests count
  - Last commit information
  - README file status
- Content script for GitHub page analysis
- Manifest V3 compatibility
- Privacy-focused design (no data collection)

### Technical
- Built with Manifest V3
- Content script injection for GitHub pages
- Chrome messaging API for communication
- Responsive popup design
- Error handling and user feedback
