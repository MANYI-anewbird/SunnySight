# Privacy Policy

**Last Updated:** [Date]

## Introduction

GitHub Repo Analyzer ("we", "our", or "the Extension") is committed to protecting your privacy. This Privacy Policy explains how we collect, use, and safeguard information when you use our Chrome extension.

## Information We Collect

### Data Collection
This extension **does NOT collect, store, or transmit any personal information** to external servers. All data processing occurs locally in your browser.

### Data Processing
- The extension only reads publicly available information from GitHub repository pages that you visit
- All analysis is performed locally on your device
- No data is sent to any external servers or third parties
- No user accounts or authentication information is collected

## Permissions Used

### activeTab Permission
- **Purpose:** Allows the extension to access the current active tab when you click the extension icon
- **Usage:** Used only to read the GitHub repository page content for analysis
- **Data:** Only processes data from the page you are currently viewing

### Host Permission (github.com)
- **Purpose:** Allows the extension to inject content scripts into GitHub pages
- **Usage:** Required to extract repository information from GitHub pages
- **Data:** Only accesses publicly visible information on GitHub pages you visit

## Data Storage

- **Local Storage:** The extension does not use Chrome's local storage or any persistent storage
- **No Backend:** There is no backend server or database
- **No Tracking:** No analytics, tracking, or telemetry data is collected

## Third-Party Services

This extension does not integrate with any third-party services, APIs, or external data processors.

## Children's Privacy

This extension is not intended for users under the age of 13. We do not knowingly collect information from children.
