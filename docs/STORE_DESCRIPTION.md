### How It Works

1. **Visit any GitHub repository page**
   - Navigate to any public GitHub repository (e.g., `https://github.com/owner/repo`)

2. **Click the extension icon**
   - The extension automatically analyzes the current page

3. **View comprehensive information**
   - All repository details are displayed in an organized, easy-to-read format
